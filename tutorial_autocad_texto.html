Tutorial de AutoCAD: Comandos de Texto

Aprende a crear y formatear texto en tus dibujos

El texto es un componente esencial en los dibujos técnicos de AutoCAD. Este tutorial te guiará a través de los comandos de texto más utilizados: TEXTO DE LÍNEA ÚNICA (DT) y TEXTO DE LÍNEAS MÚLTIPLES (MT). Aprenderás a crear, formatear y editar texto de manera efectiva en tus diseños.

Consejo: Antes de comenzar, asegúrate de tener configurado el estilo de texto adecuado (STYLE) con la fuente y altura correctas para tu dibujo.
DT

TEXTO DE LÍNEA ÚNICA (DTEXT, DT)

Comando: DTEXT o DT

El comando TEXTO DE LÍNEA ÚNICA se utiliza para crear una o más líneas de texto, donde cada línea es un objeto independiente. Es ideal para etiquetas, números o notas cortas.

Método 1: Texto con Altura y Ángulo Definidos

Se especifica la altura y el ángulo de rotación antes de ingresar el texto.

Ejemplo 1.1: Etiqueta Simple

Dibuja un texto "Muro Principal" de 3 unidades de altura y 0 grados de rotación.

Activa el comando TEXTO DE LÍNEA ÚNICA (DT).
Especifica el punto inicial: 10,10.
Especifica la Altura del Texto: 3.
Especifica el Ángulo de Rotación: 0.
Ingresa el texto: Muro Principal
Presiona Enter dos veces para finalizar.
Ejemplo 1.2: Texto Inclinado

Dibuja un texto "SECCIÓN A-A" de 5 unidades de altura rotado 45 grados.

Activa el comando TEXTO DE LÍNEA ÚNICA (DT).
Especifica el punto inicial: 50,50.
Especifica la Altura del Texto: 5.
Especifica el Ángulo de Rotación: 45.
Ingresa el texto: SECCIÓN A-A
Presiona Enter dos veces para finalizar.

Método 2: Texto Justificado (Justify, J)

Se utiliza para controlar la alineación del texto respecto al punto de inserción.

Ejemplo 2.1: Etiqueta Centrada

Dibuja el texto "TÍTULO" de altura 4, centrado en el punto (30,5).

Activa el comando TEXTO DE LÍNEA ÚNICA (DT).
Escribe J (para la opción Justificar).
Escribe C o MC (para Centro o Middle Center).
Especifica el punto central: 30,5.
Especifica la Altura del Texto: 4.
Ángulo de Rotación: 0.
Ingresa el texto: TÍTULO
Presiona Enter dos veces para finalizar.
Nota sobre la justificación

AutoCAD ofrece múltiples opciones de justificación para el texto de línea única. Algunas de las más utilizadas son:

  • Izquierda (L): Alineación estándar (predeterminada).
  • Centro (C): Centra el texto horizontalmente en el punto de inserción.
  • Derecha (R): Alinea el texto a la derecha del punto de inserción.
  • Ajustar (F): Ajusta el texto entre dos puntos, modificando su escala horizontal.
  • Medio (M): Centra el texto tanto horizontal como verticalmente en el punto de inserción.
MT

TEXTO DE LÍNEAS MÚLTIPLES (MTEXT, MT)

Comando: MTEXT o MT

El comando TEXTO DE LÍNEAS MÚLTIPLES permite crear párrafos de texto con formato avanzado, ideal para notas largas, especificaciones o descripciones detalladas.

Método 1: Texto de Párrafo (Definición de Esquina)

Se define un cuadro de texto (rectángulo) para delimitar el ancho del párrafo.

Ejemplo 2.2: Párrafo Estándar

Dibuja un párrafo de 15 unidades de ancho con texto de 2 unidades de altura.

Activa el comando TEXTO DE LÍNEAS MÚLTIPLES (MT o MTEXT).
Especifica la primera esquina: 10,60.
Especifica la esquina opuesta (para definir el ancho de 15): 25,50.
En el editor, cambia la Altura del Texto a 2.
Ingresa el texto: "Las especificaciones técnicas se encuentran en el anexo A. Toda medida es en metros."
Haz clic fuera del cuadro para finalizar.

Método 2: MTEXT con Columnas y Formato

Dibuja una nota general que use viñetas y se alinee a la derecha.

Ejemplo 2.3: Texto con Formato Avanzado
Activa MTEXT (MT).
Define el cuadro de texto (ancho de 20).
En el editor, en la pestaña Columnas, selecciona Estático y 2 columnas.
Ingresa la lista con formato:
NOTAS DEL DIBUJO Requerimientos: • Punto 1 • Punto 2 • Punto 3 FIN
Consejo: Para alinear texto a la derecha en MTEXT, selecciona el texto y usa el botón de alineación a la derecha en la barra de herramientas de formato, o usa el código \A2; al principio del párrafo.
Formato avanzado en MTEXT

El editor de MTEXT te permite aplicar diferentes formatos a tu texto:

  • Negrita, cursiva y subrayado
  • Cambiar el tipo y tamaño de fuente
  • Crear listas con viñetas o numeradas
  • Ajustar el interlineado
  • Insertar símbolos especiales
  • Crear tablas y columnas

Para acceder a estas opciones, haz doble clic en el texto MTEXT para abrir el editor.

ACTIVIDADES PRÁCTICAS

Actividad 1: Etiqueta de Símbolo (DT Justificado y Polar)

Objetivo: Colocar una etiqueta centrada y rotada sobre un círculo, simulando un símbolo.

Dibuja un CÍRCULO (C) de radio 5 en (50,50).
Activa el comando TEXTO DE LÍNEA ÚNICA (DT).
Escribe J y luego C para justificación Centro.
Especifica el punto de inserción: 50,50 (centro del círculo).
Especifica la Altura del Texto: 4.
Especifica el Ángulo de Rotación: 120 grados.
Ingresa el texto: T-120
Ver Solución (Ubicación y Entidad)

1. La entidad debe ser TEXT (Texto de Línea Única).

2. Al girar el texto a 120 grados, la línea de base del texto debe estar inclinada hacia la esquina superior izquierda.

3. El punto de inserción (50,50) debe coincidir con el centro geométrico de la entidad T-120.

4. Asegúrate de que el texto esté en una capa con un color que contraste con el círculo para mejor visibilidad.

Actividad 2: Tabla de Notas (MTEXT y Alineación)

Objetivo: Crear una tabla de notas de dos columnas usando el control de MTEXT.

Dibuja un RECTÁNGULO (REC) de 25x15 unidades para servir como contenedor.
Activa el comando MTEXT (MT) y define un área dentro del rectángulo.
Configura el ancho de columna para que ocupe aproximadamente 20 unidades de ancho.
Ingresa el siguiente contenido con formato:
NOTAS DEL DIBUJO Requerimientos: 1. Revisar escalas. 2. Usar capa MOB. FIN
Ver Solución (Tipos de Formato Aplicados)

1. NOTAS DEL DIBUJO debe usar formato de Negrita y Subrayado (dentro del editor de MTEXT).

2. La alineación de "FIN" se logra seleccionando solo esa palabra o línea y usando la justificación Derecha dentro del editor.

3. Los números de los requisitos se pueden crear como una lista numerada automática o manualmente.

4. Asegúrate de dejar una línea en blanco antes de "FIN" para mejor presentación.

RETOS DE COMPRENSIÓN

RETO 1: La Edición Rápida (DDEDIT)

Dibuja un TEXTO DE LÍNEA ÚNICA (DT) y un TEXTO DE LÍNEAS MÚLTIPLES (MT). Ejecuta el comando DDEDIT (DD) y haz clic en cada entidad de texto.

Verifica: ¿El editor que aparece es el mismo para ambas entidades?

Solución: No es el mismo editor. Al usar DDEDIT en un texto de línea única (DT), aparece un editor de texto simple en la línea de comandos. En cambio, al usarlo en un texto de líneas múltiples (MT), se abre el editor de texto enriquecido con todas las opciones de formato.

RETO 2: La Rotación Engañosa

Dibuja un TEXTO DE LÍNEA ÚNICA (DT) que diga "Cuidado" a 0 grados. Luego, utiliza el comando GIRAR (RO) para rotarlo 90 grados.

Verifica: ¿El texto se ve exactamente igual que si se hubiera creado directamente a 90 grados? ¿Cómo se diferencia la rotación por el comando GIRAR de la rotación intrínseca al comando DT?

Solución: No es exactamente igual. Cuando usas el comando GIRAR, el texto gira como un objeto completo, incluyendo su punto de inserción. Cuando especificas el ángulo directamente en el comando DT, el texto se crea con esa orientación desde el principio. La diferencia principal está en cómo se maneja el punto de inserción y la justificación del texto. Con GIRAR, todo el objeto gira alrededor del punto base, mientras que con la rotación en DT, el texto se genera ya con esa orientación.

RETO 3: Propiedades y Bloqueo

Crea un MTEXT con varias líneas de texto. Abre la ventana de PROPIEDADES (PR) y busca la sección COLUMNAS. Intenta cambiar la altura del texto y el espaciado de las líneas.

Pregunta: ¿Cómo afecta el parámetro Ancho (Width) en las propiedades a la forma en que se ajusta el texto si el valor es 0?

Solución: Cuando el ancho (Width) de un MTEXT se establece en 0, el texto se comporta como una línea única que se extiende indefinidamente hacia la derecha, sin ajuste de línea automático. Solo se ajusta la altura del texto según el valor especificado. El texto no se ajustará al ancho del cuadro de texto, sino que formará una sola línea horizontal larga. Para que el texto se ajuste automáticamente al ancho del cuadro de MTEXT, debes especificar un valor de ancho mayor que 0.

Conclusión

Los comandos de texto en AutoCAD son herramientas poderosas que te permiten añadir información clara y profesional a tus dibujos. Dominar tanto el TEXTO DE LÍNEA ÚNICA (DT) como el TEXTO DE LÍNEAS MÚLTIPLES (MT) te permitirá:

  • Crear etiquetas precisas para elementos de tu dibujo
  • Agregar notas y especificaciones detalladas
  • Mantener la consistencia en tus diseños
  • Mejorar la legibilidad y comprensión de tus planos
  • Ahorrar tiempo con opciones de formato avanzado
Práctica recomendada: Crea estilos de texto (STYLE) para diferentes usos en tus dibujos, como títulos, etiquetas y notas. Esto te permitirá mantener la consistencia en todos tus proyectos y ahorrar tiempo al no tener que formatear cada texto individualmente.
