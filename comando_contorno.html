Comando CONTORNO (BOUNDARY) en AutoCAD

Aprende a crear polilíneas y regiones cerradas a partir de áreas definidas por objetos existentes

📚 Introducción al Comando CONTORNO

El comando CONTORNO (BOUNDARY, BO) es una herramienta poderosa en AutoCAD que permite crear polilíneas o regiones cerradas a partir de áreas definidas por objetos existentes. Sus principales usos incluyen:

  • Convertir áreas delimitadas por líneas, arcos o círculos en polilíneas cerradas
  • Crear regiones para operaciones booleanas o cálculos de propiedades
  • Preparar áreas para sombreados (HATCH) o cálculos de área precisos
  • Simplificar geometrías complejas en entidades manejables
Nota: A diferencia del comando PEDIT (Editar Polilínea), el comando CONTORNO puede crear polilíneas cerradas incluso cuando los objetos originales no están conectados perfectamente.

🔧 Métodos de Trazado

Método 1: Designar Puntos Internos (Pick Internal Points)

Este método permite crear un contorno seleccionando un punto dentro de un área cerrada. AutoCAD detecta automáticamente los límites más cercanos para crear la nueva entidad.

Ejemplo 1.1: Creación de Contorno a partir de Líneas Intersectantes

Objetivo: Crear una polilínea cerrada a partir de cuatro líneas que forman un cuadrado.

  1. Dibuja cuatro LÍNEAS (L) separadas para formar un cuadrado:
    • De (0,0) a (10,0)
    • De (10,0) a (10,10)
    • De (10,10) a (0,10)
    • De (0,10) a (0,0)
  2. Activa el comando CONTORNO (BO).
  3. En el cuadro de diálogo, asegúrate de que el Tipo de Objeto esté configurado como Polilínea.
  4. Haz clic en el botón Designar Puntos Internos.
  5. Haz clic dentro del área del cuadrado.
  6. Presiona Enter para finalizar el comando.
  7. Intenta seleccionar el borde: ahora tienes una polilínea de contorno (además de las cuatro líneas originales).
Consejo: Para verificar que se ha creado correctamente, mueve la nueva polilínea. Deberías poder seleccionar todo el contorno como una sola entidad.

Ejemplo 1.2: Contorno en Área Intersectada (El Agujero)

Objetivo: Crear un contorno que represente el área de un rectángulo menos un círculo interior.

  1. Dibuja un RECTÁNGULO (REC) de 20x10 unidades.
  2. Dibuja un CÍRCULO (C) de radio 3 unidades en el centro del rectángulo.
  3. Activa el comando CONTORNO (BO).
  4. Asegúrate de que el Tipo de Objeto sea Polilínea.
  5. Haz clic en Designar Puntos Internos.
  6. Haz clic en el área entre el rectángulo y el círculo.
  7. Presiona Enter para finalizar.
Resultado: Se creará una polilínea que representa el perímetro exterior del rectángulo con un "agujero" circular. Esta es una polilínea con una isla interior.

Método 2: Usando el Tipo de Objeto "Región" (Region)

Este método crea regiones 2D en lugar de polilíneas. Las regiones son útiles para operaciones booleanas y cálculos de propiedades de masa.

Ejemplo 2.1: Creación de Región Simple

Objetivo: Crear una región a partir de un polígono.

  1. Dibuja un POLÍGONO (POL) de 6 lados:
    • Escribe POL y presiona Enter.
    • Especifica el número de lados: 6.
    • Especifica el centro del polígono: 0,0.
    • Selecciona la opción Inscrito en círculo (I).
    • Especifica el radio: 5.
  2. Activa el comando CONTORNO (BO).
  3. En el cuadro de diálogo, cambia el Tipo de Objeto a Región.
  4. Haz clic en Designar Puntos Internos.
  5. Haz clic dentro del polígono.
  6. Presiona Enter para finalizar.
Verificación: Selecciona el polígono original y la nueva región. Son dos entidades separadas. La región es un objeto 2D cerrado que puede usarse para operaciones booleanas o cálculos de propiedades.

Ejemplo 2.2: Región para Unión/Sustracción

Objetivo: Crear regiones a partir de un rectángulo y un triángulo superpuestos para operaciones booleanas.

  1. Dibuja un RECTÁNGULO (REC) de 20x10 unidades.
  2. Dibuja un TRIÁNGULO que se superponga al rectángulo:
    • Usa el comando LINE o POL (Polígono con 3 lados).
    • Crea un triángulo que cubra aproximadamente la mitad superior del rectángulo.
  3. Activa el comando CONTORNO (BO).
  4. Configura el Tipo de Objeto como Región.
  5. Haz clic en Designar Puntos Internos.
  6. Haz clic dentro del rectángulo (fuera del triángulo) y presiona Enter.
  7. Repite el proceso para crear una región del triángulo.
Operaciones Booleanas: Ahora puedes usar los comandos UNION, SUBSTRACT o INTERSECT con estas regiones para crear formas más complejas.

🎯 Actividades Prácticas

Actividad 1: El Contorno de la Línea y el Arco

Objetivo: Crear un contorno a partir de una entidad mixta (Línea, Arco, Línea) que se une con otra línea.

  1. Dibuja una POLILÍNEA (PL) con los siguientes segmentos:
    • Inicio en (10,10)
    • Segmento recto hasta (20,10)
    • Arco hasta (25,15)
    • Segmento recto hasta (10,20)
  2. Dibuja una LÍNEA que cierre la figura, yendo de (10,20) a (10,10).
  3. Activa el comando CONTORNO (BO).
  4. Configura el Tipo de Objeto como Polilínea.
  5. Haz clic en Designar Puntos Internos.
  6. Haz clic dentro del área cerrada.
  7. Presiona Enter para finalizar.
  8. Borra las tres entidades originales (las dos líneas y la polilínea).

Explicación de la Solución:

Después de completar estos pasos, solo debería quedar una única entidad en el dibujo: la Polilínea de Contorno generada por el comando BOUNDARY.

Verificación:

  1. Selecciona la entidad resultante.
  2. Verifica en las propiedades que sea una Polilínea.
  3. Observa que tiene los mismos vértices que la figura original, pero ahora es una sola entidad cerrada.

Este ejercicio demuestra cómo el comando CONTORNO puede crear una polilínea cerrada a partir de múltiples entidades que juntas forman un área cerrada, incluso cuando incluyen diferentes tipos de segmentos (rectos y curvos).

Actividad 2: Contorno del Centro (Área Anidada)

Objetivo: Crear contornos para diferentes áreas definidas por círculos concéntricos.

  1. Dibuja tres CÍRCULOS (C) concéntricos:
    • Centro en (50,50)
    • Radios: 10, 15 y 20 unidades
  2. Activa el comando CONTORNO (BO).
  3. Configura el Tipo de Objeto como Polilínea.
  4. Haz clic en Designar Puntos Internos.
  5. Haz clic dentro del círculo más pequeño (radio 10).
  6. Haz clic en el área entre el círculo de radio 15 y el de radio 20.
  7. Presiona Enter para finalizar.

Explicación de la Solución:

Al completar estos pasos, se crearán dos polilíneas de contorno:

  1. Primer contorno: Una polilínea circular que coincide con el círculo más pequeño (radio 10).
  2. Segundo contorno: Una polilínea en forma de anillo que representa el área entre los círculos de radio 15 y 20.

Observaciones importantes:

  • Cada clic en un área cerrada crea un contorno independiente.
  • El orden de selección de las áreas no afecta el resultado final.
  • Puedes seleccionar múltiples áreas en una sola ejecución del comando.

Este ejercicio demuestra cómo el comando CONTORNO puede manejar áreas anidadas y crear múltiples contornos en una sola operación.

🧩 Retos de Comprensión

Reto 1: Descomponer el Contorno

Dibuja un rectángulo usando el comando RECTANG (que ya crea una polilínea cerrada). Luego, ejecuta el comando CONTORNO (BO) y haz clic dentro de ese rectángulo.

Preguntas:

  1. ¿Se crea un nuevo contorno?
  2. ¿Por qué sí o por qué no?

Explicación:

1. ¿Se crea un nuevo contorno?
No, no se crea un nuevo contorno cuando haces clic dentro de un rectángulo creado con el comando RECTANG.

2. ¿Por qué?
El comando RECTANG ya crea una polilínea cerrada, que es el mismo tipo de objeto que crearía el comando CONTORNO. AutoCAD es lo suficientemente inteligente como para reconocer que ya existe una polilínea cerrada que define ese límite, por lo que no crea una duplicada.

Conclusión: El comando CONTORNO es más útil cuando trabajas con formas que no son ya polilíneas cerradas, como conjuntos de líneas, arcos o círculos que juntos forman un área cerrada.

Reto 2: El Sombreado y el Contorno

Dibuja una forma compleja (varias líneas que se cruzan formando una forma cerrada). Luego:

  1. Usa el comando SOMBRADO (H) para rellenar la forma.
  2. Usa el comando CONTORNO (BO) para crear una polilínea de contorno.

Pregunta: ¿Cuál de los dos comandos es más rápido para obtener una Polilínea cerrada que se necesita para calcular el área?

Explicación:

El comando CONTORNO (BO) es generalmente más rápido y directo para crear una polilínea cerrada cuando necesitas calcular el área de una forma compleja. Aquí está el por qué:

  1. CONTORNO (BO):
    • Crea directamente una polilínea cerrada en un solo paso.
    • La polilínea resultante puede usarse inmediatamente con el comando AREA.
    • No requiere pasos adicionales como podría requerir el sombreado.
  2. SOMBRADO (H):
    • Crea un patrón de sombreado que llena el área pero no crea una polilínea directamente.
    • Para obtener una polilínea del sombreado, necesitarías usar comandos adicionales como BOUNDARY o HATCHGENERATEBOUNDARY (si está disponible en tu versión).
    • Este proceso de dos pasos es menos eficiente que usar CONTORNO directamente.

Conclusión: Si tu objetivo principal es obtener una polilínea cerrada para calcular el área, el comando CONTORNO (BO) es la opción más eficiente.

Reto 3: La Región para el Área

Dibuja un polígono de 5 lados. Luego:

  1. Usa el comando CONTORNO (BO) para crear una REGIÓN a partir de él.
  2. Usa el comando AREA sobre la Región creada.

Pregunta: ¿Por qué es más confiable calcular el área de un objeto de tipo Región (o Polilínea) que de un conjunto de líneas sueltas?

Explicación:

Calcular el área de una Región o una Polilínea cerrada es más confiable que hacerlo con un conjunto de líneas sueltas por varias razones importantes:

  1. Precisión:
    • Las regiones y polilíneas cerradas son entidades únicas que definen claramente un área cerrada.
    • AutoCAD puede calcular con precisión el área encerrada sin ambigüedades.
  2. Automatización:
    • Con el comando AREA, puedes seleccionar una región o polilínea cerrada y obtener su área inmediatamente.
    • Con líneas sueltas, necesitarías seleccionar cada vértice manualmente, lo que es propenso a errores.
  3. Ventajas de las Regiones:
    • Las regiones tienen propiedades físicas como área, perímetro y centroide que se pueden consultar fácilmente.
    • Pueden usarse en operaciones booleanas (unión, resta, intersección) para crear formas más complejas.
    • Mantienen la precisión incluso con formas muy complejas.
  4. Ventajas de las Polilíneas:
    • Las polilíneas cerradas también permiten cálculos de área precisos.
    • Son más ligeras en términos de recursos que las regiones.
    • Pueden editarse fácilmente con comandos como PEDIT.

Conclusión: Tanto las regiones como las polilíneas cerradas ofrecen una forma confiable de calcular áreas en AutoCAD, eliminando los errores humanos asociados con la selección manual de puntos y proporcionando resultados precisos independientemente de la complejidad de la forma.

💡 Consejos y Trucos Avanzados

1. Uso de la Tolerancia de Separación

Cuando trabajas con formas que tienen pequeñas aberturas o imperfecciones, la opción Tolerancia de Separación en el cuadro de diálogo de CONTORNO puede ser muy útil:

  • Aumenta este valor para permitir que AutoCAD cierre automáticamente pequeños huecos en los límites.
  • Útil cuando trabajas con dibujos escaneados o importados que no son perfectamente cerrados.
  • Ten cuidado de no establecer un valor demasiado alto, ya que podría causar resultados inesperados.

2. Combinación con Otros Comandos

El comando CONTORNO es particularmente poderoso cuando se combina con otros comandos de AutoCAD:

  • HATCH: Crea contornos primero, luego aplica sombreados a las áreas definidas.
  • AREA: Usa las polilíneas generadas para cálculos de área precisos.
  • REGION/UNION/SUBTRACT: Crea regiones complejas a partir de formas simples.
  • EXTRUDE/PRESSPULL: Extruye contornos cerrados para crear objetos 3D.

3. Optimización del Rendimiento

Para dibujos complejos con muchos contornos:

  • Usa capas para organizar los contornos generados.
  • Considera congelar o desactivar capas innecesarias antes de ejecutar el comando.
  • Para áreas muy grandes o complejas, trabaja en secciones más pequeñas.

4. Exportación y Uso en Otros Programas

Las polilíneas y regiones creadas con CONTORNO pueden ser útiles más allá de AutoCAD:

  • Exporta las polilíneas a programas de diseño o ingeniería que acepten formatos como DXF o DWG.
  • Usa las regiones para cálculos de propiedades físicas en software de análisis estructural o mecánico.
  • Convierte las regiones a sólidos 3D para modelado avanzado.
Importante: Recuerda que las regiones son objetos 2D. Si necesitas trabajar en 3D, primero crea las regiones y luego extruye o usa el comando PRESSPULL para darles volumen.
