Comando Línea en AutoCAD

Aprende a dominar el comando más fundamental de dibujo en AutoCAD

El comando LÍNEA es una de las herramientas más básicas y esenciales en AutoCAD. Te permite crear segmentos de línea recta entre dos puntos en el espacio de dibujo. Aprender a usarlo correctamente es fundamental para cualquier proyecto de diseño.

Ejemplos Prácticos

Ejemplo 1: Dibujar un cuadrado Mostrar

Dibujando un cuadrado de 100x100 unidades

Presiona L y luego Enter para activar el comando LÍNEA.
Haz clic en cualquier punto de la pantalla para establecer el primer punto.
Escribe @100,0 y presiona Enter.
Escribe @0,100 y presiona Enter.
Escribe @-100,0 y presiona Enter.
Escribe C y presiona Enter para cerrar la figura.

Ejemplo 2: Dibujar una estrella Mostrar

Creando una estrella de 5 puntas

Activa el comando LÍNEA (L + Enter).
Haz clic en un punto de inicio.
Escribe @100<72 y presiona Enter.
Escribe @100<216 y presiona Enter.
Escribe @100<0 y presiona Enter.
Escribe @100<144 y presiona Enter.
Escribe @100<288 y presiona Enter.
Presiona Enter para finalizar el comando.

Ejercicios para Practicar

Ejercicio 1: Casa Simple Mostrar

Instrucciones:

Crea un dibujo de una casa simple usando solo el comando LÍNEA.

Dibuja un rectángulo de 150x100 unidades para la estructura principal.
Agrega un triángulo equilátero en la parte superior para el techo.
Incluye una puerta de 30x60 unidades.
Añade dos ventanas de 20x20 unidades.

Solución:

Activa el comando LÍNEA (L + Enter).
Dibuja el rectángulo base: haz clic en un punto y usa @150,0, luego @0,100, @-150,0 y por último C para cerrar.
Para el techo: desde la esquina superior izquierda del rectángulo, dibuja una línea a @75,50 y luego a la esquina superior derecha.
Para la puerta: desde un punto en la base (ej. @40,0), dibuja hacia arriba 60 unidades, luego a la derecha 30 unidades, y luego hacia abajo 60 unidades.
Para las ventanas: repite un procedimiento similar en las posiciones deseadas.
Ejercicio 2: Letras con Líneas Mostrar

Instrucciones:

Crea las letras "A" y "C" usando solo líneas rectas.

Cada letra debe tener 100 unidades de altura.
Usa el comando LÍNEA para todos los trazos.
Asegúrate de que las letras sean proporcionales.

Solución para la letra "A":

Dibuja una línea diagonal hacia arriba y a la derecha: @50,100
Desde el mismo punto inicial, dibuja hacia arriba y a la izquierda: @-50,100
Dibuja una línea horizontal en la mitad para la barra central.

Solución para la letra "C":

Comienza en la parte superior derecha y dibuja hacia la izquierda.
Dibuja hacia abajo en el lado izquierdo.
Dibuja hacia la derecha en la parte inferior.

Retos de AutoCAD

Reto 1: El Puente Imposible Mostrar

¿Puedes dibujar un puente que desafíe la gravedad?

Crea un diseño de puente usando solo líneas rectas. El reto es hacer que parezca que algunas partes flotan en el aire, creando una ilusión óptica interesante.

Pista: Experimenta con líneas que se cruzan en ángulos inusuales para crear la ilusión de profundidad. Usa líneas diagonales que converjan en puntos de fuga para dar la sensación de perspectiva.

Solución:

Comienza dibujando dos líneas paralelas horizontales para la base del puente.
Agrega líneas diagonales que se extiendan hacia arriba desde los extremos, creando la ilusión de distancia.
Conecta estas líneas con travesaños horizontales que parezcan más pequeños a medida que se alejan.
Añade detalles adicionales como barandillas y soportes para completar la ilusión.
Reto 2: La Ciudad del Futuro Mostrar

¿Cómo se vería una ciudad futurista hecha solo con líneas?

Crea un skyline de una ciudad futurista usando solo el comando LÍNEA. Incluye rascacielos, puentes y elementos arquitectónicos interesantes.

Pista: Usa líneas en diferentes ángulos para crear edificios con diseños únicos. Experimenta con formas geométricas interesantes para los techos y ventanas.

Reto 3: El Laberinto Misterioso Mostrar

¿Puedes crear un laberinto que desafíe a cualquiera que intente resolverlo?

Diseña un laberinto complejo usando solo líneas rectas. Asegúrate de que tenga una entrada, una salida y múltiples caminos posibles, pero solo uno que lleve a la salida.

Pista: Comienza con el perímetro exterior, luego agrega paredes internas. Asegúrate de que siempre haya al menos un camino desde la entrada hasta la salida. Usa líneas paralelas para crear los pasillos.
