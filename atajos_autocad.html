Atajos de Teclado en AutoCAD

Acelera tu flujo de trabajo con estos comandos esenciales

Dominar los atajos (alias) en la línea de comandos de AutoCAD es fundamental para mejorar tu productividad. Un atajo es una abreviatura de una o dos letras que ejecuta un comando completo, permitiéndote trabajar de manera más rápida y eficiente.

Consejo profesional: Puedes personalizar los atajos de teclado en AutoCAD usando el comando ALIASEDIT o editando el archivo acad.pgp.

1. Atajos de Dibujo

Comando Atajo Descripción
LINEA L Dibuja una línea recta
CÍRCULO C Dibuja un círculo
POLILINEA PL Dibuja una secuencia conectada de segmentos
RECTANG REC Dibuja un rectángulo
ARCO A Dibuja un arco circular
POLÍGONO POL Dibuja un polígono regular
Ejemplo Práctico: Dibujar un cuadrado usando atajos Mostrar
  1. Escribe L y presiona Enter (activa el comando LÍNEA).
  2. Haz clic en un punto de inicio en el área de dibujo.
  3. Mueve el cursor a la derecha y escribe 10 + Enter (línea horizontal de 10 unidades).
  4. Mueve el cursor hacia arriba y escribe 10 + Enter (línea vertical).
  5. Mueve el cursor a la izquierda y escribe 10 + Enter (línea horizontal).
  6. Escribe C + Enter para cerrar el cuadrado.

2. Atajos de Modificación

Comando Atajo Descripción
COPIAR CP Duplica objetos
DESPLAZAR M Mueve objetos
GIRAR RO Rota objetos
SIMETRÍA MI Crea una copia reflejada
ESCALA SC Cambia el tamaño de los objetos
RECORTA TR Recorta objetos en un límite
Ejemplo Práctico: Crear una matriz de objetos Mostrar
  1. Dibuja un círculo con C + Enter.
  2. Selecciona el círculo y escribe CP + Enter (comando COPIAR).
  3. Especifica el punto base (centro del círculo).
  4. Escribe M + Enter para modo múltiple.
  5. Mueve el cursor a la derecha y escribe 20 + Enter (distancia entre copias).
  6. Repite el paso anterior para crear más copias.
  7. Presiona Enter para terminar.

Actividades Prácticas

Actividad 1: Crear un patrón de hexágonos Mostrar

Objetivo: Usar los atajos para crear un patrón de hexágonos.

  1. Usa POL + Enter para dibujar un hexágono.
  2. Especifica 6 lados y el centro del hexágono.
  3. Usa CP + Enter para copiarlo.
  4. Crea un patrón copiando el hexágono en las 6 direcciones.
Consejo: Usa el modo Ortopolar (F8) para mantener las líneas rectas.
Ver Solución
  1. Dibuja el primer hexágono con POL + Enter, 6 lados.
  2. Usa CP + Enter y selecciona el hexágono.
  3. Selecciona un vértice como punto base.
  4. Usa M + Enter para múltiples copias.
  5. Copia el hexágono a cada vértice adyacente.
Actividad 2: Crear una llave inglesa Mostrar

Objetivo: Usar comandos de dibujo y modificación para crear una llave inglesa.

  1. Usa REC para el mango.
  2. Usa C para la cabeza circular.
  3. Usa TR para recortar el exceso de líneas.
  4. Usa F (empalme) para redondear las esquinas.
Ver Solución
  1. Dibuja un rectángulo de 60x20 para el mango.
  2. Dibuja un círculo de radio 20 en un extremo.
  3. Dibuja un hexágono inscrito en el círculo.
  4. Usa TR + Enter dos veces para recortar.
  5. Usa F + Enter, luego R + Enter y especifica un radio de 5.

Retos de Comprensión

Reto 1: El Reloj Analógico Mostrar

¿Puedes crear un reloj analógico usando solo atajos de teclado?

Instrucciones:

  1. Usa C para el borde del reloj.
  2. Usa L para las marcas de las horas.
  3. Usa RO para rotar las manecillas.
  4. Usa MI para crear marcas simétricas.
Pista: Usa la matriz polar para las marcas de los minutos.
Ver Solución
  1. Dibuja un círculo grande con C.
  2. Dibuja una línea vertical desde el centro hacia arriba.
  3. Usa AR (matriz) + Enter.
  4. Selecciona la línea y presiona Enter.
  5. Escribe PO (polar) + Enter.
  6. Selecciona el centro del círculo.
  7. Especifica 12 elementos y 360 grados.
Reto 2: El Laberinto Mostrar

Crea un laberinto usando solo comandos de línea y modificación.

Instrucciones:

  1. Usa L para dibujar las paredes.
  2. Usa O (offset) para crear caminos paralelos.
  3. Usa TR para crear pasadizos.
  4. Usa F para redondear esquinas.
Ver Solución
  1. Dibuja un rectángulo grande para el borde exterior.
  2. Usa O + Enter y especifica un desplazamiento de 10 unidades.
  3. Repite el proceso para crear varios niveles.
  4. Usa TR + Enter dos veces para recortar aberturas.
  5. Asegúrate de que haya un camino desde la entrada hasta la salida.
Reto 3: El Engranaje Perfecto Mostrar

Crea un engranaje con dientes perfectamente espaciados.

Instrucciones:

  1. Usa C para el círculo base.
  2. Usa DIV para dividir la circunferencia.
  3. Usa AR para el patrón de dientes.
  4. Usa TR para limpiar las líneas sobrantes.
Consejo: Usa el comando DIVIDE para marcar puntos de división iguales en la circunferencia.
Ver Solución
  1. Dibuja un círculo con C.
  2. Usa DIV + Enter y selecciona el círculo.
  3. Especifica 24 divisiones (para 12 dientes).
  4. Dibuja un diente en una de las divisiones.
  5. Usa AR + Enter, selecciona el diente.
  6. Elige PO (polar) y selecciona el centro.
  7. Especifica 12 elementos y 360 grados.
