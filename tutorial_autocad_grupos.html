Comando GRUPO (GROUP) en AutoCAD

Domina el uso de grupos para organizar y manipular conjuntos de objetos de manera eficiente

El comando GRUPO (G) en AutoCAD permite agrupar temporalmente un conjunto de entidades para que puedan ser seleccionadas y manipuladas conjuntamente (moverse, copiarse, girarse), sin necesidad de convertirlas en un Bloque. A diferencia de los Bloques, los grupos son más flexibles y no afectan la definición del dibujo.

Métodos de Creación y Manejo

Método 1: Creación de Grupo Nombrado (GROUP, G)

Se asigna un nombre específico al grupo, lo que permite activarlo, desactivarlo y seleccionarlo fácilmente en el futuro.

Ejemplo 1.1: Grupo de Mobiliario de Oficina

Objetivo: Dibujar un escritorio y una silla, y agruparlos.

  1. Dibuja un RECTANGULO (Escritorio) y un CÍRCULO (Silla).
  2. Activa el comando GRUPO (G).
  3. Escribe N (para seleccionar la opción Nombre / Name).
  4. Ingresa el nombre: OFICINA_1.
  5. Selecciona el escritorio y la silla. Presiona Enter.
Verificación: Haz clic en el escritorio. La silla también debe seleccionarse automáticamente.
Ejemplo 1.2: Control de Selección de Grupo

Objetivo: Desactivar la selección de grupo para editar un elemento individual.

  1. Utiliza el grupo "OFICINA_1" del ejemplo anterior.
  2. Presiona la combinación de teclas Ctrl + H (o usa el comando GROUPDISPLAYMODE).
  3. Intenta seleccionar el escritorio. Solo el escritorio debe seleccionarse.
  4. Utiliza el comando DESAGRUPAR (UNGROUP) o presiona Ctrl + H de nuevo para reactivar la selección de grupo.

Método 2: Creación de Grupo No Nombrado

Se crea un grupo automáticamente con un nombre genérico (Ej: *A1, *A2).

Ejemplo 2.1: Agrupación Rápida de Componentes

Objetivo: Dibujar un conjunto de líneas de ejes y agruparlos sin nombre.

  1. Dibuja tres LINEAS cortas y paralelas.
  2. Activa el comando GRUPO (G).
  3. Selecciona las tres líneas. Presiona Enter.
  4. El grupo se crea automáticamente (Ej: *A1).
  5. Utiliza el comando MOVER (M). Selecciona una de las líneas: todo el grupo se moverá.
Ejemplo 2.2: Añadir Entidades a un Grupo Existente

Objetivo: Añadir un nuevo círculo al grupo no nombrado.

  1. Dibuja un CÍRCULO cercano al grupo de líneas del ejemplo anterior.
  2. Activa el comando EDITARGRUPO (GROUPEDIT, GRED).
  3. Selecciona el grupo no nombrado (*A1).
  4. Escribe A (para la opción Añadir / Add).
  5. Selecciona el CÍRCULO. Presiona Enter.
Verificación: Haz clic en cualquier línea: el círculo también debe seleccionarse.

Actividades Prácticas

ACTIVIDAD 1: El Conjunto de Iluminación

Objetivo: Crear un grupo a partir de objetos y otro grupo a partir de la copia del primero, reordenando los elementos.

Instrucciones:

  1. Dibuja un CÍRCULO (Lámpara) y un TEXTO "L1".
  2. Grupo 1: Agrupa el círculo y el texto bajo el nombre LAMPARA_BASE.
  3. Copia: Copia el grupo 'LAMPARA_BASE' 10 unidades a la derecha.
  4. Grupo 2: Agrupa el objeto original y la copia bajo el nombre ILUMINACION_GENERAL.
  5. Edición: Utiliza EDITARGRUPO sobre 'ILUMINACION_GENERAL' y REMOVER el texto "L1" de la copia.
Ver Solución (Estado Final de la Entidad)
  1. Al finalizar, el grupo 'ILUMINACION_GENERAL' contendrá 4 entidades (Círculo y Texto original, Círculo copiado).
  2. La clave es que, después de REMOVER el texto de la copia, solo el círculo copiado permanece como una entidad separada dentro del grupo, mientras que el texto de la copia vuelve a ser una entidad libre (o es eliminado, dependiendo de si se selecciona remover y luego borrar).
ACTIVIDAD 2: El Reordenamiento de Componentes

Objetivo: Desagrupar un objeto, editar una parte y reagruparlo inmediatamente.

Instrucciones:

  1. Dibuja un RECTANGULO y dos LINEAS internas. Agrupa todo bajo el nombre PLACA_ELECTRONICA.
  2. Desagrupar: Utiliza el comando DESAGRUPAR (UNGROUP) sobre el grupo 'PLACA_ELECTRONICA'.
  3. Edición: Mueve el RECTANGULO 5 unidades hacia arriba (Y).
  4. Reagrupar: Vuelve a agrupar los tres elementos (el rectángulo movido y las dos líneas) bajo el mismo nombre 'PLACA_ELECTRONICA'.
Ver Solución (Verificación de Nombre)

La clave es que UNGROUP elimina el grupo, liberando las entidades. El comando GROUP posterior debe recrear el grupo con las entidades en su nueva posición. El nombre 'PLACA_ELECTRONICA' se puede reutilizar sin problemas.

Retos de Comprensión

RETO 1: Grupo vs. Bloque (Propiedades)

¿Sabías que los grupos y bloques se comportan de manera diferente en AutoCAD? ¡Descubre cómo!

Instrucciones:

  1. Dibuja un círculo.
  2. Conviértelo en un GRUPO llamado 'Grupo_A'.
  3. Dibuja otro círculo.
  4. Conviértelo en un BLOQUE llamado 'Bloque_B'.
  5. Utiliza el comando LISTA (LIST, LI) sobre cada entidad.

Pregunta: ¿Qué tipo de entidad reporta AutoCAD para el GRUPO y qué tipo reporta para el BLOQUE?

Ver Solución

El grupo sigue listando las entidades originales (en este caso, un círculo), mientras que el bloque es una 'REFERENCIA A BLOQUE' que apunta a la definición del bloque 'Bloque_B'.

RETO 2: La Selección de Entidades no Agrupadas

¿Cómo distingues entre objetos agrupados y no agrupados? ¡Averígualo con este reto!

Instrucciones:

  1. Crea un grupo de 3 círculos.
  2. Dibuja un cuarto círculo cerca, pero sin agrupar.
  3. Ejecuta el comando SELECCIONARGRUPO (SELECTGROUP) y selecciona el grupo.

Preguntas:

  1. ¿Se selecciona solo el grupo?
  2. ¿Cómo puedes diferenciar visualmente las entidades agrupadas de las que no lo están durante la selección?
Ver Solución
  1. Sí, el comando SELECTGROUP selecciona solo los objetos que pertenecen al grupo especificado.
  2. Las entidades agrupadas suelen mostrar una caja de límite de selección alrededor de todo el grupo, mientras que los objetos no agrupados muestran sus propios límites de selección individuales.
RETO 3: Agrupar en Capa Diferente

¡Los grupos pueden contener objetos de diferentes capas! ¿Cómo afecta esto a su visualización?

Instrucciones:

  1. Crea una capa A_Grupo (Color Verde).
  2. Crea una capa B_Contenido (Color Rojo).
  3. Dibuja un RECTANGULO en A_Grupo.
  4. Dibuja un CÍRCULO en B_Contenido.
  5. Agrupa ambos elementos bajo el nombre 'Doble_Capa'.

Preguntas:

  1. Si el grupo es seleccionado, ¿qué color muestra?
  2. ¿Y si se mueve el grupo a una tercera capa?
Ver Solución

Los grupos no tienen color propio. Cada entidad dentro del grupo mantiene las propiedades de su capa original, independientemente de la capa en la que se encuentre el grupo. Por lo tanto:

  1. Al seleccionar el grupo, el rectángulo se mostrará en verde y el círculo en rojo, según sus capas respectivas.
  2. Si mueves el grupo a una tercera capa, los objetos individuales conservarán sus colores originales (verde y rojo) porque heredan las propiedades de sus capas, no de la capa del grupo.
