Comandos de Utilidad/Medida en AutoCAD

Domina las herramientas de medición y consulta para verificar dimensiones, áreas y propiedades de objetos

Los comandos de Utilidad e Información te permiten verificar dimensiones, áreas, volúmenes y propiedades de los objetos sin acotar. Esto es fundamental para la validación y el cálculo de presupuestos.

Métodos de Medida

Método 1: Medición de Distancia y Radio (DISTANCE, DI / MEASUREGEOM)

Se utiliza para obtener la distancia recta entre dos puntos y los datos de entidades circulares.

Ejemplo 1.1: Distancia entre Puntos Inclinados

Objetivo: Medir la distancia real entre dos puntos en diagonal.

  1. Dibuja LINEA A: de (10,10) a (20,15).
  2. Activa el comando DISTANCIA (DI).
  3. Primer punto: (10,10).
  4. Segundo punto: (20,15).
Resultado esperado: La línea de comandos mostrará la distancia real (aprox. 11.18) y los valores Delta X (10) y Delta Y (5).
Ejemplo 1.2: Medición de Radio y Circunferencia

Objetivo: Medir el radio, diámetro y circunferencia de un círculo.

  1. Dibuja un CÍRCULO (C) de radio 5.
  2. Activa el comando MÉTODO (MEASUREGEOM, ME).
  3. Selecciona la opción Radio (Radius).
  4. Haz clic en el círculo.
Resultado esperado: Debe mostrar el Radio (5.00) y el Diámetro (10.00).

Método 2: Medición de Área (AREA, AA / MEASUREGEOM)

Se utiliza para calcular el área de una figura cerrada (polilíneas, círculos, o puntos seleccionados).

Ejemplo 2.1: Área por Puntos (Método Tradicional)

Objetivo: Medir el área de un triángulo dibujado con líneas individuales.

  1. Dibuja un TRIÁNGULO con tres LINEAS (L) separadas: (0,0) a (10,0) a (5,8) a (0,0).
  2. Activa el comando ÁREA (AREA, AA).
  3. Haz clic en los tres vértices (0,0), (10,0), (5,8).
  4. Presiona Enter.
Resultado esperado: El área calculada debe ser 40.00 unidades cuadradas.
Ejemplo 2.2: Área de un Objeto (Método Rápido)

Objetivo: Medir el área de un polígono cerrado con un solo clic.

  1. Dibuja un RECTANGULO (REC) de 15x10.
  2. Activa ÁREA (AREA, AA).
  3. Escribe O (para seleccionar Objeto / Object).
  4. Haz clic en el borde del rectángulo.
Resultado esperado: Debe mostrar el Área (150.00) y el Perímetro (50.00).

Método 3: Listado de Propiedades de Entidades (LIST, LI)

Se utiliza para obtener toda la información detallada de una entidad (coordenadas, longitud, área, capa, etc.).

Ejemplo 3.1: Coordenadas de Vértices

Objetivo: Obtener la lista de coordenadas de los vértices de una polilínea.

  1. Dibuja una POLILINEA (PL) de (0,0) a (5,5) a (10,0).
  2. Activa el comando LISTA (LIST, LI).
  3. Selecciona la Polilínea.
  4. Presiona Enter.
Resultado esperado: Se mostrarán las coordenadas X, Y, Z de cada vértice, la longitud total y la capa.
Ejemplo 3.2: Propiedades Detalladas de Bloque

Objetivo: Obtener las propiedades de inserción de un bloque.

  1. Inserta un BLOQUE (I) en (50,50), rotado 45 grados.
  2. Activa LISTA (LIST, LI).
  3. Selecciona el bloque.
  4. Presiona Enter.
Resultado esperado: Se mostrará el nombre del bloque, la coordenada de inserción (50,50) y el ángulo de rotación (45.00).

Actividades Prácticas

ACTIVIDAD 1: Cálculo de Material Restante

Objetivo: Calcular el área neta de una placa después de perforarla.

Instrucciones:

  1. Dibuja la PLACA (RECTANGULO) de 40x30 en (0,0).
  2. Dibuja la PERFORACIÓN (CÍRCULO) de radio 5 en el centro (20,15).
  3. Paso A (Área de la Placa): Usa ÁREA (AA) y selecciona la placa.
  4. Paso B (Sustracción): Escribe A (para Añadir) y luego S (para Sustraer).
  5. Selecciona el CÍRCULO. Presiona Enter dos veces.
Ver Solución (Área Neta Final)
  1. Área de la placa: 40 × 30 = 1200
  2. Área del círculo: π × 5² ≈ 78.54
  3. Área Neta Final: 1200 - 78.54 = 1121.46 unidades cuadradas
ACTIVIDAD 2: Verificación de Pendiente Inclinada

Objetivo: Calcular la longitud de proyección horizontal de un elemento inclinado usando LISTA.

Instrucciones:

  1. Dibuja una LINEA (L) de (10,5) a (30,15).
  2. Medición A (LISTA): Usa LISTA (LI) y selecciona la línea. Anota el valor Delta X (DX).
  3. Medición B (DISTANCIA): Usa DISTANCIA (DI) y haz clic en los extremos de la línea. Anota el valor Distancia.
Ver Solución (Valores Esperados)

Coordenadas: (10, 5) a (30, 15)

  1. Valor DX (Proyección Horizontal): 20.00 (30 - 10)
  2. Valor DY (Proyección Vertical): 10.00 (15 - 5)
  3. Distancia (Real): √(20² + 10²) = √500 ≈ 22.36 unidades

Retos de Comprensión

RETO 1: Área de Polilínea con Ancho

¿Sabías que el ancho de una polilínea no afecta su área? ¡Compruébalo!

Instrucciones:

  1. Dibuja una POLILINEA (PL) de 10x10.
  2. Usa el comando PEDIT (PE) para darle un ancho global (Global Width) de 5.
  3. Utiliza el comando ÁREA (AREA, AA) y selecciona la polilínea.

Pregunta: ¿El comando ÁREA calcula el área del centro de la polilínea (100) o incluye el área adicional del ancho (5) que le diste?

Ver Solución

El comando ÁREA siempre calcula el área basada en el contorno central de la polilínea, ignorando completamente el ancho de línea. Por lo tanto, el resultado será 100 unidades cuadradas (10×10), independientemente del ancho que le hayas asignado.

RETO 2: Medición de Ángulos

¿Cómo mide AutoCAD los ángulos entre líneas? ¡Descúbrelo con este reto!

Instrucciones:

  1. Dibuja dos líneas que se crucen formando un ángulo de 30 grados.
  2. Activa el comando MÉTODO (MEASUREGEOM, ME).
  3. Selecciona la opción Ángulo (Angle).
  4. Haz clic en las dos líneas.

Preguntas:

  1. ¿Qué valor de ángulo muestra AutoCAD?
  2. ¿Cómo se vería afectado el resultado si hubieras utilizado el comando LISTA (LI)?
Ver Solución
  1. AutoCAD mostrará el ángulo más pequeño entre las dos líneas (30° o 150°).
  2. El comando LISTA (LI) mostraría los ángulos de cada línea individual con respecto al eje X, pero no el ángulo entre ellas. Para obtener el ángulo entre dos líneas, MEASUREGEOM es la herramienta más directa.
RETO 3: El Perímetro y la Polilínea

¿El método de dibujo afecta las mediciones de perímetro? ¡Investígalo!

Instrucciones:

  1. Dibuja un cuadrado de 10x10 usando cuatro líneas separadas.
  2. Usa el comando ÁREA (AA) y selecciona los cuatro vértices.
  3. Dibuja un segundo cuadrado de 10x10 usando una sola POLILINEA (REC).
  4. Usa el comando ÁREA (AA) y selecciona la opción Objeto (O).

Pregunta: ¿El valor del PERÍMETRO es idéntico en ambos métodos?

Ver Solución

Sí, el perímetro será idéntico (40 unidades) en ambos casos. La diferencia está en cómo se crearon los objetos, no en sus propiedades geométricas fundamentales. Sin embargo, trabajar con una polilínea cerrada es más eficiente porque:

  1. Es un solo objeto en lugar de cuatro
  2. Facilita la selección y modificación
  3. Permite aplicar propiedades globales como ancho de línea o extrusión
